# react-query vs swr

reactQuery

```
전역상태 를 건드리지 않고 react 어플리케이션의 data 를 패치(fetch) 캐시(cache)를 하고 update 를한다.
```

SWR

```
data 의 패칭(fetching)을 위한 리액트 훅
```

이 두개의 라이브러리들의 공통점에 대해서 얘기를 해보겠다.

1.query

2.caching

3.polling

4.parallel Queries

5.initial Data

6. window focus re-fetching

7.network status re-fetching
